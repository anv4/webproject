ABOUT THE GUIDE:

This guide was inspired by my current Web Systems course at NJIT and started in July 2014. I decided to make an informative guide on the basics of HTML and CSS in hopes that someone might find this useful and may learn something new. It is meant to give basic instruction on how one may make their own basic web page.
Most of the information found for this guide is available at the World Wide Web Consortium website as well as Code Academy which teaches HTML and CSS.
To further your knowledge on the given subjects, please visit their websites: W3schools and Code Academy as they will cover the topics in more detail.
