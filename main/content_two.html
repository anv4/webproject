GETTING STYLED: CSS

What exactly is CSS?

According to the w3schools.com definition, put simply CSS (or Cascading Style Sheets) is the language (or code) that works with markup languages (in our case HTML) to provide format and style of a document. While HTML can provide colors and fonts to each individual piece of text, if you have multiple paragraphs it would take up a lot of time.
One style sheet can be shared across multiple web pages. For instance, check out the different pages of Amelia's Guide. Notice the background color and various fonts stay the same? This is all under one stylesheet maintaining the format and style.
Say you just move into a house and bought a bunch of stuff to put inside it (like furniture, appliances, etc.) but they were scattered about with no organization. You or an interior designer may decide to everything inside and organize and style in an aesthetically pleasing and simple to navigate way. Think of the stuff as content or HTML and the design you did to make the house look nicer and more organized as the CSS.

How do I position my divs?

There are different ways to position things in CSS. There is Normal Flow, Floats, and Absolute positioning, Static Positioning, Relative positioning, and Fixed positioning.
Normal flow "includes block formatting of block boxes, inline formatting of inline boxes, relative positioning of block or inline boxes, and positioning of run-in boxes."
Floats start as "a box is first laid out according to the normal flow, then taken out of the flow and shifted to the left or right as far as possible. Content may flow along the side of a float."
Absolute positioning is when "a box is removed from the normal flow entirely (it has no impact on later siblings) and assigned a position with respect to a containing block." [1]
Fixed positioning "is a subcategory of absolute positioning. The only difference is that for a fixed positioned box, the containing block is established by the browser window size. A fixed element does not move when a web page is scrolled as all other elements do."
Static Positioning "is applied by the declaration "position: static". This places the element in the normal flow. Since normal flow is the default it is not normally necessary to explicitly use this. Where it is useful is over-riding another rule of lower specifity." [2]
Relative positioning is when "the element remains in the normal layout flow but the the element's position can be specified by top, right, bottom and left properties relative to the element's normal flow position in the document." [3]
