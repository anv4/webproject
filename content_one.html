GETTING STARTED: HTML

What exactly is HTML?

According to the w3schools.com definition, put simply HTML (or Hyper Text Markup Language) is the language (or code) that describes and formats web pages. A markup language means that HTML uses markup tags to describe a page's content. HTML provides web pages with structure. Essentially, HTML and CSS make up the foundation or building blocks of the web.

How can I make my own web page?

First thing needed is a way for your web browsers to read your html file.If you do not have notepad++, sublime, or some other fancy editor, using theplain text notepad given in most computers will do just fine. When saving your page, do not save it as a text file (.txt) but as a html file which is either (.html or .htm). When you double click on your html file, youcan open it in your favorite web browser to view!
